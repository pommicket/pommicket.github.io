Elementary Cellular Automata

This is a program which runs elementary cellular automata like Rule 110. Each cell is determined by the 3 cells above it (assuming speed of light = 1), using rules specified by the rule number. The "speed of light", (A.K.A. the range) is how quickly information travels. Each cell is determined by the 2c+1 cells where c is the "speed of light". Rule number < 222n+1
